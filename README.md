# Vernala Website

This is the Vernala website built with React, TypeScript, and Vite.

## Prerequisites

- Node.js (v18 or higher)
- pnpm (v8 or higher)

## Installation

```bash
pnpm install
```

## Running the app

```bash
pnpm run dev
```

## Building the app

```bash
pnpm run build
```

## Adding a new shadcn component

```bash
pnpm dlx shadcn@latest add <component-name>
```
